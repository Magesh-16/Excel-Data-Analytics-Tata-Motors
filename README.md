# Tata Motors Sales Analysis using Microsoft Excel

## Overview

This project analyzes **30,000+ Tata Motors sales records** using
Microsoft Excel. The goal is to transform raw sales data into meaningful
business insights through data cleaning, calculations, Pivot Tables, and
an interactive dashboard.

## Business Objective

-   Analyze sales performance across regions and dealers.
-   Measure revenue, profit, discounts, and units sold.
-   Identify trends in vehicle models, fuel types, and payment methods.
-   Build an interactive dashboard for decision-making.

## Tools Used

-   Microsoft Excel
-   Pivot Tables
-   Pivot Charts
-   Excel Formulas
-   Dashboard

## Skills Demonstrated

-   Data Cleaning
-   Data Validation
-   IF, SUM, AVERAGE, COUNTIF, XLOOKUP, VLOOKUP
-   Revenue & Profit Calculation
-   Discount Analysis

-   Pivot Tables & Charts
-   Dashboard Design
